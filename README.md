# Speech-recognizer

Приложение для распознавания речи на русском языке с использованием движка Vosk. 


Конвертирует русскую речь в текст в реальном времени через микрофон.


Предварительные условия:


Установлен Python 3.7 и выше


https://www.python.org/downloads/windows/


Выбрать: Download Windows installer (64-bit) или Download Windows installer (32-bit)


Установить зависимости из папки install
